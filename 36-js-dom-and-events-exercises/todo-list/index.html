Clear Completed

Requirements

  1. When I check an item, the label is crossed out.
  2. When I click an x next to the label, the item is deleted.
  3. When I enter an item into the textbox and press enter, it is added to the list.
  4. When I click on the button next to the top textbox, all of the items are marked completed.
  5. When I click on a label, it turns into a textbox for me to modify the item.
  6. When I provide an empty value for the textbox, the item is deleted
  7. When there are completed tasks, a "clear completed" button shows up that can remove them all.
  8. When I click "active", the todo list is filtered to show items that have not been completed.
  9. When I click "completed", the todo list is filtered to show items that are completed.
  10. When I click "all", the filter is removed.
  11. When an item is completed, the label shows how many items remain
